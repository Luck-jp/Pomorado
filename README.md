🍅 Pomodoro Timer

A simple Pomodoro Timer built using Python and Tkinter.

🎯 Features
Work / Break timer system
Short & Long breaks
Visual checkmarks for completed sessions
Reset functionality

🛠️ Tech Used
Python 🐍
Tkinter 🖥️

▶️ Run
python main.py

📁 Project Structure
pomodoro-app/
├── main.py
├── tomato.png

⏱️ Timer Logic
25 min work
5 min short break
20 min long break after 4 sessions
